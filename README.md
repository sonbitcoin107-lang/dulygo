## 🌟 Tính Năng Nổi Bật

**Minh English** không chỉ là một ứng dụng học từ vựng, mà là một hệ sinh thái học Phonics và Tiếng Anh giao tiếp toàn diện dành cho trẻ em Việt Nam.

### 🔤 Hệ Thống Phonics Đồ Sộ
- **Jolly Phonics (4-7 tuổi):** Lộ trình 42 âm cốt lõi, tích hợp **Cỗ Máy Ghép Vần (Blending Board) 🚂** tương tác vật lý siêu thú vị, giúp bé đánh vần thay vì học vẹt.
- **60 Quy Tắc Nâng Cao (8+ tuổi):** Bộ quy tắc giải mã từ vựng toàn diện nhất.
- **Gương Thần Khẩu Hình Miệng 👄:** Phân tích chi tiết vị trí Môi - Răng - Lưỡi và các lỗi sai đặc trưng của người Việt Nam. Tích hợp AI đọc ví dụ dạng Anchor Word chuẩn bản xứ.

### 🎮 Gamification Chuẩn Mực
- **Trải nghiệm như Duolingo:** Học mà chơi, chơi mà học.
- **Lộ trình bài bản:** Hơn 10+ chủ đề và 80+ bài học.
- **Hệ thống phần thưởng:** Tim (Hearts), Đá quý (Gems), Chuỗi ngày (Streak), và Cửa hàng.
- **Nhân vật đồng hành:** Cú MinhTi xinh đẹp sẽ luôn động viên và ăn mừng cùng bé bằng pháo hoa rực rỡ.

### ⚡ Công Nghệ Tối Ưu
- **PWA (Progressive Web App):** Cài đặt thẳng lên màn hình chính của iPhone / iPad / Samsung Tab y hệt App Store mà không tốn dung lượng. Chạy toàn màn hình siêu mượt.
- **Offline & Caching:** Tối ưu hóa tải dữ liệu nhanh chóng với Vite.

## 🚀 Cài Đặt Lên Màn Hình Chính (Dành cho Bé)

**Trên iPad / iPhone (Safari):**
1. Mở Safari, truy cập [minhenglish.vercel.app](https://minhenglish.vercel.app)
2. Bấm nút **Chia sẻ** (Hình vuông có mũi tên hướng lên).
3. Chọn **Thêm vào MH chính (Add to Home Screen)**.
4. Bấm **Thêm**. Biểu tượng MinhTi sẽ xuất hiện trên màn hình!

**Trên Samsung / Android (Chrome):**
1. Mở Chrome, truy cập link.
2. Bấm Menu `⋮` góc trên cùng bên phải.
3. Chọn **Thêm vào màn hình chính**.
4. Xong! Mở app và bắt đầu học.

## 👨‍💻 Dành Cho Nhà Phát Triển (Local Setup)

Ứng dụng được xây dựng bằng hệ sinh thái hiện đại: `React 18`, `Vite`, `React Router`, `Framer Motion`.

```bash
# 1. Tải mã nguồn
git clone https://github.com/sonbitcoin107-lang/minhenglish.git
cd minhenglish

# 2. Cài đặt thư viện
npm install

# 3. Khởi chạy môi trường Dev
npm run dev
```

App sẽ chạy tại `http://localhost:5173`.
