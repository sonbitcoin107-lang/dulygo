# Dulygo 🦉

> Clone Duolingo học tiếng Anh cho bé 7 tuổi — miễn phí, không cần Super!

**🌐 Link:** https://sonbitcoin107-lang.github.io/dulygo

## Tính năng

- ✅ 10 chủ đề học tiếng Anh (80+ bài)
- ✅ 4 loại bài tập: Trắc nghiệm, Sắp xếp từ, Điền vào chỗ trống, Nối đôi
- ✅ Hệ thống XP, Tim, Streak, Gems
- ✅ Mascot Duo cú xanh
- ✅ Confetti animation + hiệu ứng âm thanh
- ✅ Hồ sơ + Huy hiệu
- ✅ Cửa hàng Gems
- ✅ PWA — Add to Home Screen (trông như app thật!)
- ✅ Không có quảng cáo, không cần đăng ký, không cần Super

## Cài vào Samsung Tab

1. Mở **Chrome** trên Samsung Tab
2. Vào: `https://sonbitcoin107-lang.github.io/dulygo`
3. Bấm menu `⋮` → **"Add to Home Screen"**
4. Đặt tên "Dulygo" → Bấm **Add**
5. Mở icon từ màn hình chính → Học thôi! 🚀

## Phát triển

npm install
npm run dev
